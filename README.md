<h1>💊 NearMyMed</h1>
<p><strong>AI-Powered Medicine & Healthcare Platform</strong></p>

<p>
NearMyMed is a smart healthcare platform that helps users 
<b>find medicines, discover cheaper alternatives, locate pharmacies, and handle emergencies</b> — all in one place.
</p>

<hr>

<h2>🌐 Live Demo</h2>
<p>
<a href="https://near-my-med.vercel.app/" target="_blank">
https://near-my-med.vercel.app/
</a>
</p>

<hr>

<h2>📌 Problem Statement</h2>
<ul>
  <li>Users don’t know cheaper alternatives of medicines</li>
  <li>Difficult to find nearby pharmacies quickly</li>
  <li>Emergency situations lack fast decision support</li>
  <li>Overdependence on pharmacists</li>
</ul>

<hr>

<h2>💡 Solution</h2>
<ul>
  <li>🔍 Smart medicine search</li>
  <li>💰 Cost-effective alternatives</li>
  <li>📍 Pharmacy locator</li>
  <li>🚑 Emergency pre-booking system</li>
  <li>🤖 AI-based healthcare guidance</li>
</ul>

<hr>

<h2>✨ Features</h2>

<h3>🔎 Search System</h3>
<ul>
  <li>Search by name, salt, disease</li>
</ul>

<h3>💊 Alternatives</h3>
<ul>

  <li>Safe alternatives</li>
</ul>

<h3>🏥 Pharmacy</h3>

</ul>

<h3>🚨 Emergency</h3>
<ul>
  <li>Emergency medicine booking</li>
  <li>Ambulance & hospital support</li>
</ul>

<h3>🛒 Cart System</h3>
<ul>
  <li>Add to cart</li>
  <li>Checkout</li>
  <li>Order history</li>
</ul>

<h3>📧 Email System</h3>
<ul>

</ul>
